# Getting Started

Install the dependencies

```bash
  go mod tidy
```

Run the project

```bash
  go run main.go
```

Running backend in [http://localhost:3333](http://localhost:3333) 

## Tecnologies Documentation

[Golang](https://go.dev/doc/)
